# Code of Conduct

This project follows a simple rule: be respectful and useful.

## Expected behavior

- Be kind and constructive.
- Assume good intent, but ask for clarity when needed.
- Keep discussions focused on improving the project.
- Respect different experience levels.

## Unacceptable behavior

- Harassment or personal attacks.
- Discriminatory language.
- Spam or off-topic promotion.
- Publishing private information without permission.

If a discussion becomes unproductive, maintainers may moderate issues, pull requests, or comments.

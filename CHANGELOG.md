# Changelog

All notable changes to this project will be documented in this file.

## [0.5.0] - 2026-05-01

### Added

- Experimental MCP-style stdio server.
- `repo-readme-polisher-mcp` console entry point.
- `inspect_project` and `generate_readme_draft` tools.
- GitHub Actions release workflow for version tags.
- Release process documentation.

## [0.4.0] - 2026-05-01

### Added

- `--ai` mode that writes an AI rewrite prompt next to the generated README draft.
- Guardrailed prompt builder for honest README rewriting.

## [0.3.0] - 2026-05-01

### Added

- `--lang en|zh` for English and Chinese README drafts.
- `--template minimal|portfolio` output styles.
- Chinese sample README draft.

## [0.2.0] - 2026-05-01

### Added

- Richer project detection for frameworks, databases, testing tools, and deployment hints.
- JSON metadata output via `--format json`.

## [0.1.0] - 2026-05-01

### Added

- Initial Python CLI implementation.
- Local project scanner.
- Rule-based stack detector.
- README draft generator.
- Example output.
- Tests.
- Open-source project documentation and GitHub templates.
